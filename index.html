Participants are invited to use the file upload link below to share relevant information in preparation for the October 30th, 2015 initial consortium meeting.

The Cities and Law Schools Consortium is Convened by UMKC Law School and Brooklyn Law School.

This web page is part of an Open Innovation Network Repository for the Cities and Law Schools Consortium

This Repository hosts shared files about the programs, projects and people collaborating in the Network

This Repository is hosted on GitHub, so all information is easy to discover, re-use and contribute to or fork.

This Repository is designed and configured to enable any consortium member to set up a local Open Innovation Network in just a few clicks.
